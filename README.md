# IMDB Sentiment Analyzer
Simple Neural Network + TF-IDF for IMDB Movie Review Sentiment Analysis (Positive/Negative)

## Model
- TF-IDF text representation
- 2-layer MLP Neural Network
- Trained on imdb_balanced_10k.csv

## GitHub CI/CD
Automatically trains and uploads model to Hugging Face Hub on push.
